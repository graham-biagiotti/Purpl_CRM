🟡 Zapier Webhook Ready to Activate

Automatically push new Local Line orders to purpl CRM via Zapier — no manual exports needed.

Setup Instructions

  1. Create a new Zap in Zapier
  2. Trigger: Local Line → New Order
  3. Action: Webhooks by Zapier → POST
  4. Paste your webhook URL below into the Zapier URL field
  5. Map fields: order_id, buyer_name, products, total, date
  6. Test and activate the Zap
